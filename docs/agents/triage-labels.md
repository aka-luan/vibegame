# Triage Labels

The engineering skills use these canonical roles and exact GitHub label strings:

| Role              | GitHub label      | Meaning                                           |
| ----------------- | ----------------- | ------------------------------------------------- |
| `needs-triage`    | `needs-triage`    | A maintainer needs to evaluate the issue          |
| `needs-info`      | `needs-info`      | The issue is waiting on reporter information      |
| `ready-for-agent` | `ready-for-agent` | Fully specified and ready for an autonomous agent |
| `ready-for-human` | `ready-for-human` | Requires human implementation or judgment         |
| `wontfix`         | `wontfix`         | Will not be actioned                              |

When a skill names a canonical role, use the corresponding exact label. Do not create synonyms or apply multiple state labels simultaneously.
