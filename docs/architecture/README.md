# Architecture Decision Records

These records capture the major choices approved by `PLAN.md`. Read the records relevant to an area before changing its design.

Status values:

- **Accepted:** the current approved decision.
- **Superseded:** retained for history but replaced by a newer record.
- **Proposed:** not yet approved for implementation.

Create a new record to change a major decision. After implementation begins, do not rewrite an accepted record to disguise a reversal; mark it superseded and link its replacement.
